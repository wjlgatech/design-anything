# Garden & landscape

**Input:** site photo/plan + climate + intent → **Output:** planting/hardscape layout plans + printable hardscape elements (planters, edging, furniture).

## The stack
- **Site capture:** scene-to-layout skill (photo → structured plan); COLMAP for photogrammetry of existing terrain.
- **Printable hardscape:** the repo's golden example is literally a planter ([examples/planter](../../examples/planter/generate.py)) — the pattern scales to edging, benches, pavers.
- **Earth-material printing:** WASP TECLA lineage for garden structures.

## Governing principles
- **Olmsted's proof (P-biophilia)** — Central Park still functions as designed after 165 years; landscape is the longest-feedback-loop design domain. Design for the 20-year canopy, not the install photo.
- **Garden city kernel (Howard)** — open space is structural, not residual; reserve it first.
- **P9 climate is the client** — plant selection and hardscape thermal behavior derive from climate zone; vernacular precedent first.
- **Courtyard/prospect-refuge patterns (P5)** apply outdoors: enclosure gradients, sheltered seats facing open lawns.

## Ready gate seeds (roadmap)
Hardscape elements pass the print gate (drainage holes required for planters — a G5 candidate: containment vs drainage declared). Planting plans carry hardiness-zone evidence per species; no evidence ⇒ not specified.
