# Commercial architecture — 3D printing + AI

**Input:** program + site + budget text, massing sketch → **Output:** IFC blueprint, phased for conventional + printed construction.

## The stack
Shares the residential stack ([architecture-residential](../architecture-residential/README.md)) plus:
- **Scale players:** COBOD BODXL (Qatar), ICON Titan multi-story (to 9m), Europe's largest 3DCP project (COBOD).
- **Generative massing:** Autodesk Forma (solar/carbon analysis at concept stage).
- **Market context:** 3DCP forecast as a multibillion-dollar segment by 2030 (Apr-2026, constructconnect).

## Governing principles (deltas from residential)
- **P4 program first** — commercial form derives from circulation, egress, and tenant flexibility; style is applied to a valid skeleton.
- **Jacobs' conditions at ground level** — mixed use, short blocks, street activation scored on any multi-building output; never optimize one building in a void.
- **P8 layers matter more at scale** — tenant churn is the design condition: structure (60y) ≠ services (15y) ≠ fit-out (5y).
- **Egress and accessibility tables are blocking gates**, not advisories.

## The anti-portfolio warning
Katerra raised ~$2B to vertically integrate commercial construction and died in 6 years. The lane is **tools and blueprints that arm builders** — never becoming the builder.
